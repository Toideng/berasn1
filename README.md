# Naive implementation of BER's ASN.1 length encoding

### Building on Linux:

```
chmod u+x build
./build
```

To start server:
```
./server
```

To start client:
```
./client
```

### Building on Windows:

First open `cmd.exe` and navigate to this directory. Then:

```
build.bat
```

To start server:
```
./server.exe
```

To start client:
```
./client.exe
```
